Detailed Description

QPSK Demapper (hard decision)

Inverzni korak QPSK mapiranja korištenog u PBCHEncoder.qpsk().

Mapiranje u encoderu:
    I = 1 - 2*b0
    Q = 1 - 2*b1

Inverzno (hard decision):
    b0 = 0 ako I > 0, inače 1
    b1 = 0 ako Q > 0, inače 1
